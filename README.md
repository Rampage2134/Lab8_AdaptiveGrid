# Лабораторная работа №8. Адаптивная верстка: Grid

**Выполнена основная цель работы:** освноение современной технологии CSS-разметки  Grid.


**CSS Grid** — это мощная двумерная система компоновки, которая позволяет
создавать сложные сетки по строкам и столбцам.

1. **Grid идеально подходит для:**
    * Создания полноценных макетов страниц
    * Сложных раскладок с несколькими областями
    * Галерей изображений
    * Дашбордов и административных панелей

2. **Основные концепции:**
    * **Grid-контейнер** — родительский элемент с display: grid
    * **Grid-элементы** — дочерние элементы внутри контейнера
    * **Grid-линии** — разделительные линии между ячейками
    * **Grid-ячейки** — пересечение строки и столбца
    * **Grid-области** — прямоугольные области из нескольких ячеек
___
### Общая идея

|**Критерий**|**Flexbox**|**CSS Grid**|
|:----------|:----------:|:----------:|
|**Тип компоновки**|Одновреемнная|Двумерная|
|**Управление**|Строка **или** стобец|Строки **и** столбцы|
|**Основные назначения**|Выравнивание элементов|Построение макетов|
|**Подходит для**|Компонентов|Страниц и сеток|

**Вывод:** Flexbox лучше подходит для **линейных структур**, Grid - для **табличных и сложных**
___
